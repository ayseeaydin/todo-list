# TaskFlow – Basic JavaScript Todo App

TaskFlow, temel JavaScript pratiği amacıyla geliştirilmiş modern bir görev yönetimi uygulamasıdır. ES6 yapıları, DOM işlemleri, event handling ve localStorage kullanımı üzerine odaklanır.

## Özellikler
- Görev ekleme, silme ve tamamlama
- Öncelik seçimi (Low / Medium / High)
- Aktif, tamamlanmış ve tüm görevler için filtreleme
- İstatistik gösterimi (toplam, aktif, tamamlanan)
- localStorage ile kalıcı görev saklama
- Bootstrap 5 tabanlı basit modern arayüz

## Teknolojiler
- HTML5
- CSS3 (custom + Bootstrap 5)
- Vanilla JavaScript (ES6)

## Çalıştırma
1. Proje klasörünü bilgisayarınıza indirin.
2. `index.html` dosyasını bir tarayıcıda açın.
3. Her şey tarayıcı içinde localStorage ile çalışır; ek bir kurulum gerekmez.

## Amaç
Bu proje; DOM manipülasyonu, event yönetimi, array işlemleri, template rendering ve localStorage mantığını pekiştirmek için hazırlanmıştır.
